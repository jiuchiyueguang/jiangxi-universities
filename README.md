# 江西高校信息查询网站

> 🎓 为准大一新生打造的江西省前60所高校信息查询平台

## 功能特点

- ✅ **每次打开弹出引导弹窗**：QQ群信息（群号：340865393），一键复制
- ✅ **实时搜索**：支持按校名、城市、专业等关键词搜索（防抖优化）
- ✅ **多维筛选**：按城市（11个）、学校层次（8种）筛选
- ✅ **详情弹窗**：点击"查看详情"查看完整信息
- ✅ **响应式设计**：适配手机、平板、电脑
- ✅ **安全加固**：XSS防护、CSP策略、输入过滤、外链安全
- ✅ **键盘快捷键**：`Ctrl+K` 或 `/` 聚焦搜索，`Esc` 关闭弹窗

## 部署方式

### 方式一：本地直接打开
双击 `index.html` 即可在浏览器中查看。

### 方式二：GitHub Pages（推荐 - 免费）
1. 在 GitHub 创建新仓库
2. 将 `江西大学` 文件夹内所有文件上传到仓库
3. Settings → Pages → Source 选择 `main` 分支 → Save
4. 几分钟后即可通过 `https://你的用户名.github.io/仓库名` 访问

### 方式三：Vercel（推荐 - 免费）
1. 注册 [Vercel](https://vercel.com)
2. 导入 GitHub 仓库或直接拖拽文件夹部署
3. 自动获得 `https://项目名.vercel.app` 域名

### 方式四：Netlify（推荐 - 免费）
1. 注册 [Netlify](https://netlify.com)
2. 将整个文件夹拖拽到部署区域
3. 自动获得 `https://项目名.netlify.app` 域名

### 方式五：任意静态服务器
将文件夹内所有文件上传到任意 Web 服务器的根目录即可（Nginx/Apache/Caddy 等）。

## 安全特性

| 措施 | 说明 |
|------|------|
| XSS 防护 | 全部使用 `textContent` 渲染用户输入，不使用 `innerHTML` |
| CSP 策略 | 严格的内容安全策略，禁止内联脚本 |
| 输入过滤 | 搜索输入去除 HTML 标签和危险字符 |
| 外链安全 | 所有外链添加 `rel="noopener noreferrer"` |
| URL 验证 | 仅允许 https/http 协议的链接 |
| 防抖优化 | 搜索输入 250ms 防抖，避免恶意高频请求 |
| 原生 dialog | 使用浏览器原生 `<dialog>` 元素，内置焦点管理和键盘支持 |

## 技术栈

- HTML5（原生 `<dialog>` 弹窗）
- CSS3（Grid/Flexbox 响应式布局，CSS 变量）
- Vanilla JavaScript（无框架依赖，零外部库）
